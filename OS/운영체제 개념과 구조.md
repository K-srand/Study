컴퓨터 시스템 구성 정의
CPU : 명령어를 실행하는 하드웨어
Processor : 하나 이상의 CPU를 포함하는 물리적 칩
Core : CPU 내부의 독립적인 연산 처리 단위
Multicore : 두 개 이상의 독립 코어를 단일 직접회로로 이루어진
